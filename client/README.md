# Aramark UTSC — Labour Scheduling System

AI-powered scheduling prototype built to enforce CBA compliance,
eliminate hour miscalculations, and ensure seniority fairness
under UNITE HERE Local 75 Collective Agreement.

## Tech Stack
- Frontend: React 18 + Vite + Tailwind CSS + Zustand
- Backend: Node.js + Express + SQLite
- AI: Anthropic Claude API
- Deployment: Vercel (client) + Railway (server)
